# 喻小灵

## 简介

这是一个基于uni-app实现的工具类App，目前实现的功能有：

- 随机短视频播放
- 抖音/快手视频直链解析下载
- 今日新闻
- 快递/物流信息查询
- 有道翻译
- IP信息查询
- 网站测速
- 爱情文案生成
- 舔狗日记
- 骚扰电话查询
